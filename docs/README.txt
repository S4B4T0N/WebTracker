# WebTracker GitHub Pages package

Obsah:
- index.html — hlavná prezentačná stránka
- privacy-policy.html — samostatná Privacy Policy
- styles.css — spoločné štýly
- assets/ — ikona a screenshoty

## Nasadenie na GitHub Pages
1. Skopíruj celý obsah tohto priečinka do koreňa repozitára `WebTracker` alebo do samostatného branch/priečinka pre Pages.
2. Commitni a pushni zmeny.
3. Na GitHub otvor `Settings -> Pages`.
4. Ako Source nastav branch s týmito súbormi.
5. Po publikovaní bude stránka dostupná na URL v tvare:
   `https://tvoj-username.github.io/WebTracker/`

## Čo ešte upraviť
- V `privacy-policy.html` doplň kontakt.
- V `index.html` môžeš doplniť odkaz na GitHub repozitár alebo Chrome Web Store.
- V priečinku `assets/` môžeš nahradiť ukážkové screenshoty vlastnými finálnymi obrázkami aplikácie.
